# SimpleCam

Камера на Camera2 / MediaCodec. Пре-запись в кольцевой буфер. Аппаратный EIS.

## Сборка APK через GitHub Actions

1. Создать репозиторий на GitHub (публичный или приватный).
2. Загрузить все файлы из этого архива в корень репозитория.
3. Перейти во вкладку **Actions → Build SimpleCam APK → Run workflow**.
4. После завершения (~3–5 минут) открыть запуск и скачать артефакт **SimpleCam.apk**.
5. Передать APK на телефон и установить (нужно включить «Установка из неизвестных источников» в настройках Android).

## Hardware EIS

Чекбокс **HW EIS** в панели ⚙. Автоматически скрывается (серый) если устройство не поддерживает аппаратную стабилизацию. Нельзя переключать во время записи — кнопка блокируется.

## Требования

Android 8.0+ (API 26), Camera2, разрешения CAMERA + RECORD_AUDIO.
